5) Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason?

6) What are the tags required to create a navigation list in HTML?
<nav> is the root tag for a navigation list, followed by <ul> for an unordered list, <ol> for an ordered (numbered) list, and <dl> for a descriptive list. Within unordered and ordered lists, entries on the list are identified with the tag <il>; for descriptive lists, it is <dd>.
